# Bootstrap LEVEL 2

## CSS样式

- 图片（img-）
- 颜色
    * 文本颜色（text-）
    * 背景颜色（bg-）
- 定位
    * 浮动（pull-）
    * 块居中（center-block）
    * 显示和隐藏（show、hide）


## 组件
- 警告框与关闭
- 下拉菜单与三角（dropdown、caret）
- Glyphicons字体图标（glyphicon glyphicon-）
- 标签页（nav nav-tabs）
- 按钮组（btn-group）
    * 按钮组尺寸
    * 按钮组嵌套
    * 垂直按钮组
    * 两端对齐排列的按钮组
- 表单（form-group）
- 输入框组（input-group）
- 导航（nav-pills，即标签页）
